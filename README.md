# sentiment-anlyze
Sistem ini mengklasifikasikan sentimen kata sifat Bahasa Indonesia menggunakan IndoBERT. Data diberi label otomatis dengan kata kunci positif, negatif, dan netral, lalu model dilatih dengan weighted loss untuk menangani ketidakseimbangan kelas. Hasil akhirnya diekspor sebagai model siap pakai untuk aplikasi web.
